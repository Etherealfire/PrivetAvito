# PrivetAvito
Тестовое задание для стажёра

### 2 Task
В качестве второго задания я выбрал 2.1.

## Описание
Данный проект содержит ручные и автоматизированные тесты для API микросервиса объявлений.

В рамках задания были выполнены:
- составление тест-кейсов для API;
- автоматизация части тест-кейсов;
- запуск автотестов;
- фиксация найденных дефектов.

---

## Используемые технологии
- Python 3
- Pytest
- Requests

---

## Подготовка окружения

1. Установить Python версии 3.9 или выше
2. Установить зависимости:

```bash
pip install -r requirements.txt
```
--- 

 Структура проекта
```bash
qa-internship-tests/
│
├── all_tests.py       # Автоматизированные тесты API
├── requirements.txt   # Зависимости проекта
├── 2Task_tests.md     # Ручные тест-кейсы
├── BUGS.md            # Найденные дефекты
└── README.md          # Инструкция по запуску
```

Для запуска всех автотестов выполнить команду:

```bash
pytest all_tests.py -v
```
